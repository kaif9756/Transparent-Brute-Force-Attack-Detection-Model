# 🛡️ Brute Force Attack Detection Model

## 📌 Overview
This project detects brute force login attempts using Machine Learning techniques by analyzing authentication logs.

## 🚀 Features
- Detects multiple failed login attempts
- Identifies suspicious IP addresses
- Alerts when login threshold exceeded
- Log-based anomaly detection

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Linux Log Analysis

## 📊 How It Works
1. Reads authentication logs
2. Extracts failed login attempts
3. Applies ML classification model
4. Generates alert for suspicious behavior

```bash
pip install -r requirements.txt
python main.py
